# 星星之火
简化模型：将目前创业内容抽象成三个实体
  
1. 土壤 -- 人力  
2. 水 -- 资金  
3. 种子 -- 项目  
   
![image](https://github.com/user-attachments/assets/6b05c67e-c8e9-44af-9245-99948099d7f5)

最终目的，向土里放不同种子。孵化能存活的，维护它们。  

目前重心只在土壤和种子阶段
步骤：
1. 筹集人力形成土壤。准备创建一个组织，吸引和管理成员。
   1.1 组织层级，组织规范，成员共同维护
   1.2 通讯联系方式
   1.3 组织和成员的利益，组织的目标
   1.3 成员分类，成员审核等

2. 有大概100个以上的成员开始找赛道。
