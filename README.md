# 星星之火
简化模型：将目前创业内容抽象成三个实体
  
1. 土壤 -- 人力  
2. 水 -- 资金  
3. 种子 -- 项目  
   
![image](https://github.com/user-attachments/assets/6b05c67e-c8e9-44af-9245-99948099d7f5)

普通人目前创业困境：1. 没有足够的资源。2. 个人难以对抗资本。


我们想做的事，形成人力资本：
1. 构建跨领域资源网络。
2. 对加入成员的赋能。
3. 利用人力资源寻找和孵化新的项目。

第一步建立一个组织，形成土壤。
进入的成员先进行能力提升。
培训：普通人+ai 约等于一个专业人士。
